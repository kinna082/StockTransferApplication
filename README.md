# Stock Transfer App

Starter monorepo for a multi-branch stock transfer system using:

- React (frontend)
- ASP.NET Core Web API (backend)
- SQL Server (database)

## Structure

- `frontend` - React UI for store managers and admins
- `backend` - .NET solution with clean architecture layering
- `database` - SQL Server schema and seed scripts

## Next Setup

1. Create React app dependencies in `frontend`.
2. Restore and run backend solution in `backend`.
3. Run schema script in SQL Server.
